# AI-Based Recruitment Screening System

## Overview
This project is a simple AI-based recruitment screening system that helps in filtering and ranking candidates based on their skills.

It uses Natural Language Processing (NLP) techniques like **TF-IDF Vectorization** and **Cosine Similarity** to match candidate profiles with job requirements.

---

## Features
- Candidate dataset creation
- Skill-based filtering
- TF-IDF vectorization of text data
- Similarity calculation using cosine similarity
- Ranking candidates based on job fit

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

---

## Project Workflow
1. Create dataset of candidates
2. Extract and process skills
3. Convert text data into numerical form using TF-IDF
4. Calculate similarity between job description and candidate skills
5. Rank candidates based on similarity score

---

## Example Use Case
HR teams can use this system to:
- Automatically shortlist candidates
- Reduce manual screening time
- Improve hiring efficiency
